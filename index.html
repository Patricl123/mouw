Workflow

Workflow нужен для правильного работы с ветками. он помогает координировать работу авто слияние и для теста. а так же для изменений и управление в работе

Git Rebase отличие от git merge

Git merge -коммит объединяет изменения из созданой ветки и исходной ветки. Обычно используем крупных изменений .

git Rebase - мы можем изменнить текущую ветку.Это может основная или та которой мы работаем.перемещение из ветки на коммит а так же переписывает историю комитов

Git pull отличие от git fetch

git pull - используем для извлечения и загрузки содержимого из удаленного репозитория

git fetch - тоже самое можно еще и коммиты так же его считают безопасным загружает удаленное содержимое, но не обновляет рабочее состояние репозитория оствлая работу
